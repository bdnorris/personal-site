Paper Balls

Experimenting With Vue.js and My Obsession With Randomness

I was sure I could create a “stop-motion” effect on the web fairly easily. So, after shooting the photos, I made a simple Codepen for the stop motion effect.

I’m also often thinking about how randomness, something trivially easy to create in code, can be used to make things that are in flux and unpredictable. Perhaps I’m simply fascinated by it because so much of design is about putting everything in it’s right place, it’s hard to resist making things that are, at least within certain parameters, out of your control.

So, I started by writing this jQuery script to randomly move squares around the viewport. Then, wanting to learn a little more Vue.js, and in particular, use the CLI, I spun up a project and re-wrote it all in Vue.

Scroll Detection

The Educational Advantages of Writing it Yourself

While working on the idea for the first student project in DES-440, it occurred to me that a very popular affect on the web right now is animating items as they come into the viewport, and it was likely I would be asked how this was done.

A popular library for this is Wow.js. But I thought it would be useful and educational to attempt to write my own. Currently the script has a number of nice features that still need documenting, and has not really been tested on a production site. I hope to not only use it in the wild soon, but properly document it so others can use it.

It’s also dependent on jQuery. I’d like to un-jQuery soon, making it even more lightweight.

Haiku Generator

Combining Silly Phrases, Randomness and Learning Meteor.js

This project started when my friend Joey had compiled a list of haiku phrases in a Google Spreadsheet and asked if I could do anything interesting with it. This was the result.

Feeling like the idea had some legs, I thought it would be a good opportunity to learn a little Meteor.js. I created an app that would randomly pull five or seven syllable lines from a Mongo DB. Meteor made it trivially easy to add an administrative backend to enter new lines, or delete old ones.
